> 커머스 프로젝트 주요 기능 구현, DB 연동

Firebase는 NoSQL로, 2차원 테이블 형식인 스키마를 가지는 RDBMS(MySQL 등)와 달리 정해진 스키마가 없다. 따라서 RDBMS보다
자유도가 높다. 다른 프로젝트에서 Firebase cloud storage 사용했을 땐, 자유도가 높은 장점도 있지만 조작이나 기획이
까다로워진다는 생각을 하기도 했었다.

그러나 SQL 쿼리문을 직접 작성하여 데이터를 조작하지 않아도 되며, Firebase측에서 제공하는 함수와 메소드를 사용하면 편리하게
다룰 수 있다는 점이 큰 장점이라고 생각하여 사용하게 되었다.(이 외에도 Firebase 연동으로 계정(회원가입)을 관리하고, cloud storage로 데이터를 관리, store로 이미지 및 미디어 데이터 관리까지
한번에 가능하다는 점이 좋았다.)
