> Python 기본 문법 정리

기본 자료형 (숫자, 문자열, 리스트, 딕셔너리) / 변수는 값을 가리키고 있다. (값을 담는 상자)

### ❗ 자료형 (int, float, double, string 등을 상황에 맞게 변환해주는 것 잊지말기)

#### 숫자

- 사칙연산: + - \* /(몫) %(나머지)
- 제곱: a\*\*b (a의 b 제곱)
- boolean: True, False (대문자로 표기)
- 조건 연산자: ==(같을 때) !=(다를 때) ...
- 논리 연산자: and, or, not

자료형에 맞는 +를 해주어야 한다~ 자바스크립트는 자료형이 없는 언어이므로 자유로운거(그래서 Typescript 출현)

- 숫자를 문자열로 str()
- 문자열을 정수로 int()

#### 문자열

- 문자열 길이: len(문자열)
- 문자열 자르기: 문자열[n:m] (n부터 m까지~)
- 특정 문자를 기준으로 문자열 자르기: 문자열.split('특정 문자')
  특정 문자를 선택하려면 배열로 선택해주면 된다 문자열.split('.')[0] 잘린 문자열 기준 첫번째 원소 선택

#### 리스트

배열, 순서가 중요하게 값을 담는 것. 자료형에 관계없이 마음대로 담기 가능. list()

- 리스트에 원소 담기: 리스트.append(추가할 값)
- 리스트 자르기: [:]
- 리스트 선택 (인덱스값을 마이너스로 하면 거꾸로)
- 리스트 길이 구하기: len(리스트)
- 리스트 정렬: sort() 오름차순, sort(reverse=True) 내림차순
- 리스트에 원소가 들어있는지 boolean으로 반환: 원소 in 리스트
  ```py
  a=[1,2,3,4,5]
  result=100 in a
  print(result) #False
  ```

#### 딕셔너리

객체, 키, 속성 값으로 이뤄진 것
리스트와 딕셔너리가 조합되어 들어가기도 한다. (ex. 키로 문자열, 값으로 리스트, 리스트 값으로 딕셔너리)

- 딕셔너리 키가 있는지 boolean으로 반환: 키 in 딕셔너리

#### 집합(set)

집합은 원소의 중복을 제거하여 반환한다.({}로 표현됨)
집합을 사용하여 교집합, 합집합, 차집합(집합 연산자 &, |, -로)을 구할 수 있다.
a라는 리스트가 존재한다면 a_set=set(a)로 나타낼 수 있다.

```py
student_a = ['물리2','국어','수학1','음악','화학1','화학2','체육']
student_b = ['물리1','수학1','미술','화학2','체육']

a_set=set(student_a)
b_set=set(student_b)

print(a_set&b_set) #교집합
print(a_set|b_set) #합집합
print(a_set-b_set) #차집합
```

### 예외 처리문

try-except로 예외(에러) 상황이 발생했을 때 처리해주는 문법.

잡아내기 어려운 상황이 될 수 있기 때문에 웬만하면 사용하지 않는 것이 좋다.

#### 클래스

객체 지향적: 클래스에 객체와 관련된 속성을 넣어두고 컨트롤 할 수 있는 함수를 붙여주고 함수를 불러다가
각각의 인스턴스를 만들어 제어하는~

```py
class Monster(): #class Monster
    hp=100
    alive=True
    def demage(self, attack): #함수 정의
        self.hp=self.hp-attack
        if self.hp<0:
            self.alive=False
    def status_check(self): #함수 정의
        if self.alive==True:
            print('live')
        else:
            print('dead')

m1=Monster() #m1은 인스턴스.
m1.demage(150)
m1.status_check()

m2=Monster()
m2.demage(90)
m2.status_check()
```
