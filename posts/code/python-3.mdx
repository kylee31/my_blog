> 내장함수 정리

#### list(리스트)

- .count(찾을 값)
  찾는 값이 리스트 안에 몇 개 있는지 세어주는 메서드

```py
lst=[1,2,3,4,5,5]
num=lst.count(5)
print(num) #2
```

- min,max
  최대 최소 값

```py
num=[1,2,3,4,5]
a=min(num)
b=max(num)
print(a,b) #1 5
```

- index 값
  리스트에서 특정 요소의 인덱스 값 찾기. 중복되는 값이 있다면 가장 첫번째 인덱스 반환.
  index(value, start, end)

```py
num=[1,2,3,4,5]
print(num.index(4)) #3
```

- 길이가 정해진 0으로 초기화된 리스트 만들기
  for문을 사용하여 생성 가능

```py
result=[0 for i in range(n)] #길이 n인 리스트
```

- 리스트 공백으로 출력하기
  [파이썬 리스트 공백으로 구분하여 출력하기](https://bamdal.tistory.com/entry/%ED%8C%8C%EC%9D%B4%EC%8D%ACpython-%EB%A6%AC%EC%8A%A4%ED%8A%B8-%EA%B3%B5%EB%B0%B1%EC%9C%BC%EB%A1%9C-%EA%B5%AC%EB%B6%84%ED%95%98%EC%97%AC-%EC%B6%9C%EB%A0%A5%ED%95%98%EA%B8%B0)

1. 반복문 사용
   end를 지정하여 출력

```py
num=[1,0,0,1]
for i in range(len(num)):
    print(num[i],end=' ')
```

2. join사용(문자열)
   문자열 메서드인 join을 사용하기 때문에 숫자인 경우 map으로 문자열 리스트로 변경하여 사용가능

```py
num=[1,0,0,1]
print(' '.join(map(str,num)))

num=['a','b','c','d']
print(' '.join(num))
```

- 값의 합(내장함수)
  리스트나 튜플 등의 값을 모두 더해주는 내장함수

```py
num=[1,2,3,4]
print(sum(num)) #10
```

- 아스키 코드(숫자)를 문자로, 문자를 아스키 코드(숫자)로
  chr(숫자), ord(문자)

```py
print(chr(65)) # a
print(ord('a')) # 65
```
