▶ 파이썬은 이전 스타트업 근무에서도 초반에 잠깐 사용하게 되었던 일이 있어서, 알고리즘을 파이썬으로 학습하는 게 좋은 방법이라고 생각이 들었다. 기술 매니저님께서도 비슷한 맥락으로 파이썬도 계속 다뤄보는 게 좋다고 말씀해주셨다. 언제 어디서 쓰일지 모르는 언어같다.. 또 아무래도 자바스크립트보다는 알고리즘 학습에 훨씬 적합한 언어이다.

```py
#1~n까지의 합
def sum_func(n):
    if n < 1:
        return
    return n + sum_func(n - 1)
```

```py
#조합 (n개의 숫자 중 m개 조합, 중복 포함)
n, m = map(int, input().split())

def combi(n, m, start, result):
    if len(result) == m:
        print(" ".join(map(str, result)))
        return
    #중복 포함하지 않으려면 i+1로 조건 변경하기
    for i in range(start, n + 1):
        result.append(i)
        combi(n, m, i, result)
        result.pop()

combi(n, m, 1, [])
```
