#### 중복되는 부분 함수로 만들기

[백준 - 누울 자리를 찾아라](https://www.acmicpc.net/problem/1652)
문제에서 이중 배열의 행,열을 변경하여 검사하는 로직을 구현했는데, 중복되는 부분이 발생했다.
따라서 해당 코드를 함수로 만들어 매개변수로 배열을 받아 결과를 출력하도록 리팩토링하였다. (괜찮은 코드인진 모르겠다)

```py
N = int(input())
room_wid, room_hei = [input() for _ in range(N)], []  # 가로방, 세로방

for i in range(N):
    result = ""
    for j in range(N):
        result += room_wid[j][i]
    room_hei.append(result)

def check_sleep(room):
    can_sleep = 0
    for idx, line in enumerate(room):
        space = line.split("X")
        for box in space:
            if len(box) >= 2:
                can_sleep += 1
    return can_sleep

wid, hei = check_sleep(room_wid), check_sleep(room_hei)
print(wid, hei)
```

```py
n = int(input())
m = int(input())
s = input()
count, idx, result = 0, 0, 0
while idx < (m-1):
    if s[idx:idx+3] == 'IOI':
        idx += 2
        count += 1
        if count == n:
            result += 1
            count -=1
    else:
        idx += 1
        count = 0

print(result)
```

#### 좋은 문제!

[백준 - 스위치 켜고 끄기](https://www.acmicpc.net/problem/1244)
[백준 - IPv6](https://www.acmicpc.net/problem/3107)

기술 매니저님께서 1,3번 문제가 현업에서도 쓰일만한 좋은 문제 유형이라고 하셨다.
해당 문제를 다시 정리해서 깔끔하게 풀어봐야겠다. (아래는 제출 후 일부 수정한 코드)

**-** bool로 감싸주지 않아도 0,1이므로 True,False 검사 가능
**-** 20씩 잘라서 프린트하는 방법 잊지 말기! (for문은 n개만큼 반복할 수 있다)

```py
num = int(input())  # 스위치 개수
switch = list(map(int, input().split()))  # 스위치 정보
stu_num = int(input())  # 학생 수
stu_info = [list(map(int, input().split())) for _ in range(stu_num)]  # 성별,스위치 번호
for info in stu_info:
    s, n = info
    if s == 1:
        for i in range(num):
            if (i + 1) % n == 0:
                switch[i] = 0 if bool(switch[i]) else 1
    else:
        switch[n - 1] = 0 if bool(switch[n - 1]) else 1
        prev, next = n - 2, n
        while prev >= 0 and next < num and switch[prev] == switch[next]:
            switch[prev] = 0 if bool(switch[prev]) else 1
            switch[next] = 0 if bool(switch[next]) else 1
            prev -= 1
            next += 1
for i in range(0, num, 20):
    print(" ".join(map(str, switch[i : i + 20])))
```

- 정답 코드 (참고)

  ```py
  line = input()

  # 나중에 0000을 두 번 추가하지 않기 위해, 맨 앞과 뒤에 필요한 경우 미리 0000 추가하기
  if line[0] == ":":
      line = "0000" + line
  if line[-1] == ":":
      line = line + "0000"

  # :로 split한 후, 각각의 문자열을 4자리로 만들어 줄게요.
  a = line.split(":")
  k = len(a)
  ans = []
  for c in a:
      # 빈 문자열인 경우 0000을 넣어주기
      if c == "":
          # 전체 길이가 8이 되도록 0000을 넣어줘요.
          for i in range(8 - k + 1):
              ans.append("0000")
      else:
          # 문자열의 길이가 4가 되도록 앞에 0을 넣어줘요.
          ans.append("0" * (4 - len(c)) + c)
  print(":".join(ans))
  ```
