#### 중복되는 부분 함수로 만들기

[백준 - 누울 자리를 찾아라](https://www.acmicpc.net/problem/1652)
문제에서 이중 배열의 행,열을 변경하여 검사하는 로직을 구현했는데, 중복되는 부분이 발생했다.
따라서 해당 코드를 함수로 만들어 매개변수로 배열을 받아 결과를 출력하도록 리팩토링하였다. (괜찮은 코드인진 모르겠다)

```py
N = int(input())
room_wid, room_hei = [input() for _ in range(N)], []  # 가로방, 세로방

for i in range(N):
    result = ""
    for j in range(N):
        result += room_wid[j][i]
    room_hei.append(result)

def check_sleep(room):
    can_sleep = 0
    for idx, line in enumerate(room):
        space = line.split("X")
        for box in space:
            if len(box) >= 2:
                can_sleep += 1
    return can_sleep

wid, hei = check_sleep(room_wid), check_sleep(room_hei)
print(wid, hei)
```

#### 4일차 문제

[백준 - IOIOI](https://www.acmicpc.net/problem/5525)
부분 점수를 얻었던 문제이다. N ≤ 100, M ≤ 10000 조건에 한해서만 통과되는 코드였다.
(짜면서도 부분 점수일걸 알았다.. 1 ≤ N ≤ 1,000,000 / 2N+1 ≤ M ≤ 1,000,000조건에서는 시간복잡도가 어마어마해진다..)

```py
n = int(input())
m = int(input())
s = input()
count, idx, result = 0, 0, 0
while idx < (m-1):
    if s[idx:idx+3] == 'IOI':
        idx += 2
        count += 1
        if count == n:
            result += 1
            count -=1
    else:
        idx += 1
        count = 0

print(result)
```

팀원 분의 코드를 참고하여 학습했다. (IOI가 반복되는 문자열이므로 패턴을 이용하여 검사하는 방식)
